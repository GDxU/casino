How to Bet
She's a Rich Girl™ features 9 paylines. Choose to play any number of lines.

LINE BET
Displays the current bet per line shown in currency. Press the left arrow (-) to decrease the bet per line. Press the right arrow (+) to increase the bet per line.

LINES
Press the left arrow (-) to decrease the number of paylines played. Press the right arrow (+) to increase the number of paylines played.

SPIN
Press to spin the reels and submit the bet.

Auto Spin Feature
Auto Spin plays the game automatically for a selected number of spins at the current number of lines and line bet. Select the desired number of lines and line bet, press AUTO SPIN and choose how many spins to play. Auto Spin plays the specified number of spins until the STOP button is pressed, a bonus is triggered, or the balance is insufficient to spin again.

Auto Spin is only available in approved jurisdictions.

[ top ]


Free Spins Bonus
Three Bonus symbols in any position on the centre three reels trigger the Free Spins Bonus.

Triggering the Free Spins Bonus awards three initial free spins. The reels in the Free Spins Bonus have fewer symbols for more frequent wins.

During the Free Spins Bonus, the bonus can be retriggered by getting one or more Diamond Run symbols anywhere on the reels. Retriggering the Free Spins Bonus awards one additional free spin. Be prepared for a lot of extra spins as this game retriggers often. The bonus can be retriggered up to a maximum of 100 free spins per bonus.

The bet in the bonus is the same as on the reel spin that triggered the bonus.

[ top ]


Diamond Run Feature
In the Free Spins Bonus, the Current Run and Record Run meters keep track of the number of free spins. The Current Run meter displays the number of times the reels have spun so far in the current bonus. The Record Run meter displays the longest run achieved by the player across all play sessions. The maximum achievable run is 100 free spins.

Rules
Misuse or malfunction voids all pays and plays.

All line win symbols must appear on a played line and on consecutive reels, beginning with the far left reel.

Only highest winner paid on each payline. Line wins are multiplied by the line bet on the winning line.

If one or more Rich Girl logo symbols substitute in a base game win, the pay for that win is doubled.

If one or more Diamond Run symbols substitute in a free spin win, the pay for that win is doubled.

Scatter wins are multiplied by the total bet.

Coinciding wins on different paylines are added. Scatter wins are independent from payline wins and are also added to the total amount paid.

Wins are shown in currency.

Bonus symbols appear in columns 2, 3, and 4 only in the base game.

Regardless of the bet size, there is an award cap on any single transaction. See the paytable for details. A transaction includes the results of the Free Spins Bonus plus the outcome which launched the bonus. If the award cap is reached in the bonus, the bonus ends immediately irrespective of any remaining free spins.
