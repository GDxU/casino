getFullLineBonus()

getFullLineBonus(integer  $countSymbols = 5) : array

Получение данных по дополнительным выплатам по линиям

Массив символов, которые проверяются содержится в параметрах игры в lineBonus['symbols'] Массив выплат по количеству содержится в lineBonus['multiplier']

Parameters

integer $countSymbols

Returns

array —

totalMultiple - общий множитель по всем линиям, которые прошли lines - массив линий, которые прошли. Содержит lineId(номер линии), multiple(множитель) и offsets(Положение символов на слоте) win - true\false count - Количество символов для проверки. То есть длина выигрышной линии. primaryLine - ID последней выигрышной линии.

startAvalanche()

startAvalanche(array  $report, array  $addOffset = array()) : array

Если есть выигрышные линии, то символы на этих линиях убираются и заменяются предыдущими невидимыми символами слота. Репорт делается для каждой победы до тех пор, пока выигрышных линий не будет.

checkExpandWild()

checkExpandWild(array  $reels, array  $symbolsID, integer  $wild)

Проверка на возможность сделать из барабана wild-барабан.

Если отображаемые символы барабана содержат только символы, переданные в $symbolsID, то отображаемые символы заменяются на $wild и барабан становится wild-барабаном

Parameters

array $reels

Массив номеров барабанов(начинается с 0), на которых проходит проверка
array $symbolsID

Массив числовых идентификаторов символов, которые должны быть на барабане
integer $wild

Идентификатор вайлда, который заменит все отображаемые символы барабана
